# Navegação no Warp

## Navegação entre Abas

| Atalho | Ação |
|--------|------|
| ⌘ + T | Nova aba |
| ⌘ + W | Fechar aba |
| ⌘ + 1-9 | Ir para aba N |
| ⌘ + ⇧ + [ | Aba anterior |
| ⌘ + ⇧ + ] | Próxima aba |

## Navegação entre Panes

Panes são divisões dentro de uma aba:

| Atalho | Ação |
|--------|------|
| ⌘ + D | Split vertical |
| ⌘ + ⇧ + D | Split horizontal |
| ⌘ + [ | Pane anterior |
| ⌘ + ] | Próximo pane |
| ⌘ + ⇧ + Enter | Maximizar pane |

## Navegação entre Blocks

| Atalho | Ação |
|--------|------|
| ⌘ + ↑ | Block anterior |
| ⌘ + ↓ | Próximo block |
| ⌘ + ⇧ + C | Copiar output do block |
